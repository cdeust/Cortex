Graph Glossary
What each element represents
Nodes
Domain
A project or knowledge area you work in (e.g. "frontend", "devops"). Domains are the large hub nodes — everything else orbits around them. Size reflects how many sessions you've had in that domain.
Entry Point
The typical way you start working in a domain — recurring first-message patterns clustered from your session history.
Pattern
A recurring behavioral pattern detected across your sessions — keyword sequences that appear repeatedly.
Tool
A Claude Code tool you frequently use in this domain (Read, Edit, Grep, Bash, etc.). Size reflects usage ratio.
Feature
A behavioral feature learned by sparse dictionary analysis of your session patterns.
Memory
A piece of information stored by Cortex — decisions, facts, errors, or context. Memories have heat and importance.
Entity
A named thing extracted from memories — functions, files, dependencies, technologies, errors, or decisions.
Edges
Bridge
A cross-domain connection — shared patterns, vocabulary, or structural links between two different domains.
Co-occurrence
Two entities that frequently appear together in the same memories.
Causal (caused_by / resolved_by)
A cause-effect relationship between entities.
Imports / Calls
Code-level relationships — one module importing another, or one function calling another.
Biological Mechanisms
Emotional Tagging
Memories with emotional charge get colored by their dominant emotion. Emotional memories have pulsing rings.
Neuromodulation
Four chemical channels modulate how strongly memories are encoded: DA, NE, ACh, 5-HT.
Synaptic Plasticity
LTP: edges strengthen on co-appearance. LTD: inactive edges weaken. STDP: temporal ordering builds causal direction.
Quality Assessment
Quality Ring
Every node has a quality score (0–100%) based on its type-specific signals. The arc ring around each node shows quality: green ≥60% (strong signal), amber ≥30% (moderate), red <30% (likely noise). Arc length is proportional to the score.
Domains — scored by session count, confidence, connections.
Entries/Patterns — scored by frequency, confidence.
Tools — scored by usage ratio, average per session.
Features — scored by activation magnitude.
Memories — scored by heat, importance, access count, recall rank.
Entities — scored by heat, connection count.
Visual Encoding
Node size — reflects importance. Node glow — intensity reflects heat. Quality arc — colored ring shows quality score. Edge brightness — reflects weight. Flow particles — animated dots along edges.
